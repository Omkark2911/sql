# sql
This project is a simple authentication system built using Node.js and Express, with PostgreSQL as the database hosted on Neon. It provides basic REST APIs for user signup and signin, allowing users to store and retrieve their credentials securely using parameterized SQL queries. The backend is connected to a serverless Postgres database, ensuring scalability and ease of setup. This project demonstrates fundamental concepts of backend development, including routing, handling requests and responses, and integrating a database. Future improvements include implementing password hashing with bcrypt and adding JWT-based authentication for enhanced security.
